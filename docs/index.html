KeyMBO

Web Flasher
M5Stack AtomS3 • ESP32-S3 • USB HID

Flash KeyMBO from your browser

Install the latest KeyMBO firmware onto an M5Stack AtomS3 with one click. Requires Chrome or Edge (Web Serial) and a USB-C cable.

Install

AtomS3 only Chrome / Edge HTTPS required Not iOS Safari
If the device doesn’t show up, try a different cable (data-capable), unplug/replug, or hold BOOT while plugging in to enter the bootloader.

Steps

  1. Plug in the M5Stack AtomS3 via USB-C.
  2. Click Install and select the AtomS3 serial device.
  3. Wait for flashing to complete, then the device will reboot.
  4. Verify the AtomS3 enumerates as a USB HID device.
This installer flashes: bootloader, partitions, boot_app0, firmware, and LittleFS.
Having trouble? Make sure you’re on Chrome/Edge, the cable supports data, and no other app is holding the serial port.
